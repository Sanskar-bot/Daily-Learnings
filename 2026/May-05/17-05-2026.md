### Exam Preparation — Mobile Communication (Exam: May 19, 2026)

> Full day dedicated to revising Mobile Communication syllabus ahead of Tuesday's exam.

#### Unit 1 — Introduction to Mobile Communication
- Evolution of mobile networks: **1G → 2G → 3G → 4G LTE → 5G**
- Key differences between generations: speed, latency, multiplexing technique, spectrum use
- **Cellular concept** — frequency reuse, cell splitting, cell sectoring
- Co-channel interference and its relation to frequency reuse factor (N)
- **Handoff / Handover** — hard handoff vs soft handoff, when and why it occurs

#### Unit 2 — GSM (Global System for Mobile Communication)
- GSM architecture: **MS → BTS → BSC → MSC → PSTN/ISDN**
- Role of each component: Mobile Station, Base Transceiver Station, Base Station Controller, Mobile Switching Centre
- **TDMA** (Time Division Multiple Access) — how GSM divides channels by time slots (8 slots per 200 kHz channel)
- GSM frame structure: hyperframe → superframe → multiframe → frame → burst
- **SIM card** role — IMSI, authentication key (Ki), network authentication via A3/A8 algorithms
- GSM security: authentication challenge-response, TMSI (Temporary Mobile Subscriber Identity) for privacy
- Location updating, call setup, and handover procedures

#### Unit 3 — CDMA & Spread Spectrum
- **CDMA** (Code Division Multiple Access) — all users share the same frequency and time, separated by unique orthogonal codes (Walsh codes)
- **Spread spectrum** principles: Direct Sequence (DS-SS) and Frequency Hopping (FH-SS)
- Processing gain and its role in interference rejection
- **Near-far problem** in CDMA and how power control solves it
- Soft handoff in CDMA — "make before break" vs GSM's hard handoff

#### Unit 4 — Mobile Network Architecture & Protocols
- **Location management** — HLR (Home Location Register) and VLR (Visitor Location Register)
- **Roaming** — how a network authenticates a user outside their home network
- **WAP (Wireless Application Protocol)** — architecture, WAP gateway, WML
- Mobile IP — agent discovery, registration, tunnelling, triangle routing problem
- **GPRS** — packet-switched overlay on GSM, SGSN and GGSN roles

#### Unit 5 — Wireless LAN & Advanced Topics
- **IEEE 802.11 (Wi-Fi)** — CSMA/CA vs CSMA/CD, why collision avoidance (not detection) is used in wireless
- Hidden node problem and RTS/CTS mechanism as the solution
- **OFDM** (Orthogonal Frequency Division Multiplexing) — used in 4G LTE and Wi-Fi, subcarrier orthogonality
- **MIMO** (Multiple Input Multiple Output) — spatial multiplexing for increased throughput
- 5G key features: mmWave, beamforming, network slicing, ultra-low latency (< 1ms)
