*What is an Advanced Persistent Threat (APT)?*
- A state-sponsored or state-affiliated cyber espionage organization with fundamentally different operational profile from ordinary cybercriminals
- APTs operate with long-term objectives, significant resources, and sophisticated toolsets

*ChamelGang Profile:*

| Attribute | Detail |
|---|---|
| Suspected Origin | China (PRC — state-affiliated) |
| Active Since | At least 2019 |
| Primary Objectives | Espionage, data theft, critical infrastructure disruption |
| Attribution Confidence | Medium-High (based on tool signatures, TTPs, infrastructure overlap) |

*Known Targets:* Government entities in USA, Russia, Japan, Taiwan, Vietnam, Nepal — broad geopolitical targeting consistent with state objectives

| Tool / Technique | Role in the Attack |
|---|---|
| **CatB Ransomware** | Primary payload — file encryption, browser password theft, sandbox evasion |

#### Network Protocols & OSI Model — Research Started

**What are Network Protocols?**
- A set of rules that define how devices communicate over a network — covering the format, timing, and method of data exchange
- **WHAT** — defines the format and structure of data: headers, payload, error codes
- **WHEN** — controls timing and sequencing: when to send, acknowledge, or retry
- **HOW** — specifies encoding and transmission so different devices understand each other

**Types of Network Protocols:**
- **Secure Connection Protocols** — HTTPS, TLS, FTP — protect data using encryption and authentication
- **Network Management Protocols** — ICMP, Syslog, NetFlow — monitor and troubleshoot network infrastructure
- **Network Communication Protocols** — HTTP, UDP, IP — handle the core exchange of data between devices
